 1. What is Semantic HTML? 
 Semantic HTML is the correct form of using HTML in that the tags are used for their given purpose thus making content more understandable.
    2. What is HTML used for? 
    HTML is used to give structure to a webpage.
    3. What is an attribute and where do we put it?
    Attributes provide additional information about a tag and are found within the tag its specifying.

    7. What is an id? 
    An id is an attribute used to specify a particular element.
    8. What elements can I add an id to? 
    It is a Global attribute so it can be used on any element.
    9. How many times can I use the same id on a page?

    10. What is a class? 
    A class is an attribute used to specify one or more elements.
    11. What elements can I add a class to? 
    It is a Global attribute so it can be used on any element.
    12. How many times can I use the same class on a page? 
    More than once.

    15. How do I reference an id?
    With a hashtag (#) before the name of the id being referenced. (#reference {})
    16. What is the difference between a section and a div?
    The difference is div is also used as a container for HTML elements in CSS while section is used to seperate headers, footers, etc. 
    17. What is CSS used for?

    18. How to we select an element? Example - every h2 on the page
    In CSS you would use the selector h2 {}.
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
    An id can only be declared once on a page while a class can be used on more than one element. - For example, if you wanted to change the look of multiple headers at once you would use class. Whereas an id would be used to specify one element.
    20. How do we select classes in CSS?
    With a (.) before the name of the class. (.name {})
    21. How do we select a p element with a single class of “human””?

    22. What is a parent child selector? When would this be useful? 
    A parent child selector is a selector where you are selecting a specific tag under another tag. the tag under would be the child and the tag above would be the parent. it is useful when you want to select a specific tag within an element as opposed to the whole element.
    23. How do you select all links within a div with the class of sidebar?

    37. Walk me through Lambda's git flow. 
    This is how we will be submitting our projects. We will first fork the repo of the project. 
    From there we will use the command line to clone the repo to our local machine by pasting the copied clone link after the 'git clone' command. 
    We then create a branch, in which our individual work will be done, by using the command 'git checkout -b 'branch-name''. 
    Once we begin working we can save and make a log of changes made with notes. 'git add .' , 'git commit -m "comment here"'.
    With our branch name created we would go back to the github project, replace main with our branch name and then send a pull request.
